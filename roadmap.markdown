# Roadmap
This is the roadmap for the implementation of the API.

## Version 0.1
Includes the following operations:

* /task/create
* /task/list
* /task/complete

## Version 0.2
Adds the following operations:

* /time/add
* /time/start
* /time/stop

## Version 0.3
Adds the following operations:

* /category/create
* /category/list

## Version 1.0
Adds the following operations:

* /task/update
* /category/update
* /category/archive
